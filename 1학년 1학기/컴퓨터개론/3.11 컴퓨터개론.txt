 cpu : 중앙 처리 장치  => 복잡한 계산/ gpu : 그래픽( rgb, 알파값) => 단순한 정보처리

중앙 처리 장치 :

주 기억 장치 = 메모리, 램 = > 전원이 있을 때만 저장 가능

보조 기억 장치 : 하드디스크,  ssd카드(반도체 기반 저장장치)

입력 장치 : 키보드, 마우스

출력 장치 : 모니터, 프린터  CRT(대가리 큰 모니터) : 음극선 쏘아서 화면 비춤


pc :=> cpu, 보조기억장치, 주 기억 장치   메인보드(=마더보드)에 장착


중앙처리장치 : 연산, 정보처리,  트랜지스터가 많이 결합되어 있음

무어의법칙 : 반도체 칩에 집적할 수 있는 트랜지스터의 숫자는 18개월마다 두 배씩 증가한다

=> cpu + gpu 기능 다 있음

=> 한 개의 칩에 완전하게 구동되게 만들어놓음

cpu : 논리 연산과 제어 기능을 빠르게 수행 / gpu : 그래픽 연산만을 전담 / 병렬 프로세스 => 고속 연산 아주 잘함

코어 : 병렬 처리에 사용되는 핵심 연산 장치 ( 한 코어가 한순간에 하나의 착업을 처리함 )
