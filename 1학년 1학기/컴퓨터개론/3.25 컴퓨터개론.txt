3.25 컴퓨터개론


ls: 현재 디렉토리의 파일과 디렉토리 목록을 보여줍니다.
cd: 디렉토리를 변경합니다.
mkdir: 새 디렉토리를 생성합니다.
rm: 파일이나 디렉토리를 삭제합니다.
cp: 파일이나 디렉토리를 복사합니다.
mv: 파일이나 디렉토리를 이동하거나 이름을 변경합니다.
cat: 파일의 내용을 출력합니다.
grep: 파일에서 특정 패턴을 검색합니다.
chmod: 파일이나 디렉토리의 권한을 변경합니다.
chown: 파일이나 디렉토리의 소유자를 변경합니다.
ps: 현재 실행 중인 프로세스 목록을 보여줍니다.
top: 시스템 리소스 사용량을 실시간으로 모니터링합니다.
kill: 프로세스를 종료합니다.
tar: 파일이나 디렉토리를 압축하거나 해제합니다.
wget: 인터넷을 통해 파일을 다운로드합니다.
ping: 네트워크 상태를 확인합니다.
ifconfig: 네트워크 인터페이스 정보를 출력합니다.
ssh: 원격 시스템에 안전하게 접속합니다.
grep: 파일 내에서 패턴에 해당하는 행을 찾습니다.


알고리즘?

시간적사양, 공간적사양

선택정렬, 삽입정렬, 버블정렬  :   n차함수 # 노마드코드(ㅋㅋㅋㅋ)


알고리즘 : 해결해야하는 문제가 주어졌을 때, 이 문제의 해답을 구하기 위한 절차를 순서대로 명확하게 나타낸 것

정의 => 주어진 문제를 해결하기 위한 단계적인 절차

( 입력 -> 알고리즘 -> 출력 )

알고리즘 조건 : 

1입력 : 모호하지 않고 정의된 입력( 알고리즘은 0개 이상의 입력을 갖음)
 2출력 : 1개 이상의 출력이 반드시 존재
 3명확성 : 각 명령어의 의미는 모호하지 않고 명확해야함
 4. 유한성 : 반드시 종료되어야 함
5. 유효성 : 현재 실행 가능한 연산이어야함


알고리즘 기술 방법 4가지

1. 영어나 한국어와 같은 자연어로 표현
2. 흐름도(flowchart)로 표현
3. 유사 코드(pseudo-code) 사용
4. 특정한 프로그래밍 언어

ex 배열에서 최댓값을 찾는 알고리즘


(c언어 => 절차지향 /  자바 => 객체지향 )
